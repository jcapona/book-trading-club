# book-trading-club

Set up your library online and trade books with other users. Full stack JS.

Uses Node.js + MongoDB + [SailsJS](http://sailsjs.org)

## Test it

Check it out live @ [heroku/booktrader](https://booktrader.herokuapp.com/)
